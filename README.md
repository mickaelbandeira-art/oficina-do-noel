# Quiz Natalino AeC

Sistema de quiz natalino desenvolvido para a AeC (Academia e Consultoria).

## Sobre o Projeto

Este é um projeto de quiz interativo com tema natalino, desenvolvido com React, TypeScript e Vite.

## Tecnologias Utilizadas

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- React Router DOM
- Shadcn/ui Components

## Como Executar Localmente

1. Clone o repositório
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure as variáveis de ambiente no arquivo `.env`
4. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Build para Produção

```bash
npm run build
```
